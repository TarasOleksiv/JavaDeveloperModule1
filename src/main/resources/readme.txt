Структура папки "resources" та порядок виконання.

0.1) DBscheme.png - схема бази.
0.2) Перед виконанням скриптів необхідно створити чисту базу на сервері.

Порядок виконання скриптів:

Частина 1.
1) initDB.sql - створює всі необхідні таблиці на базі, створеній попередньо в пункті 0.1
2) populateDB.sql - заповнює даними таблиці створені в пункті 1.

Частина 2.
3) 01.sql - виконує завдання №1;
4) 02.sql - виконує завдання №2;
5) 03.sql - виконує завдання №3;
6) 04.sql - виконує завдання №4;
7) 05.sql - виконує завдання №5;
8) 06.sql - виконує завдання №6;
_________________________________

clearDB.sql - очищає базу від створених таблиць та в'юх.
